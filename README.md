## Installing
```bash
git clone https://github.com/RomanMilishchuk/Sakamoto_Blade
yarn install
yarn start
```
    
## Usage
   Connect localhost:3000  
   Player1 controls: **WASD** - Movements **SPACE** - Shooting  
   Player2 controls: **⬆⬅⬇➡** - Movements **J** - Shooting
